## 概要
 【ゲームタイトル】BREACH HACK
 【バージョン】1.00
 【ジャンル】ブラウザミニゲーム
 【特徴】ミッション制 / 制限時間・警告システム
 【難易度】簡単
 【プレイ時間】1プレイ3分程度
 【種別】フリーウェア
 【政策言語】HTML / CSS / JavaScript
 【ゲーム制作者】三浦颯

BREACH HACK は、JavaScriptの状態管理とイベント制御を学ぶために制作したハッキングをテーマとしたブラウザ向けミニゲームです。
プレイヤーはログを読みながら、以下の3つのミッションを順にクリアしていきます。
１．コマンド入力ミッション　２．暗号解読ミッション　３．制限時間付きキー入力ミッション


## 作成時間・作成目的
【作成時間】
- 人数：一人
- 時間：12日間
【作成目的】
- JavaScriptの基礎（状態管理・イベント処理）を学ぶため  
- HTML と CSS を用いて、UIや見た目にもこだわった作品を制作するため  

## 起動方法
- フォルダをダウンロード
- ローカルサーバを起動(JavaScript使用のため)
- ブラウザで BREACH HACK.html　を開く

## アンインストール方法
　ダウンロードしたフォルダをすべて削除してください。レジストリや外部データは使用していません。

## 遊び方
1. ミッション1：正しいコマンド順にボタンを押します  
2. ミッション2：文字を順に押して、正解の単語を作ります  
3. ミッション3：制限時間内に指定された文字を入力します  
   （タイムオーバーでゲームオーバー）

※ 全ミッション共通で、失敗すると警告が加算され、3回失敗するとゲームオーバーとなります。

## 使用技術
- HTML : 画面設計
- CSS : UI表現
- JavaScript :状態管理・ゲーム進行ロジック

## 工夫点・改善点

【工夫点・こだわった点】
- ミッションごとに処理を分離し、保守性を意識しました  
- 状態(進行状況や警告数)を一元管理し、ゲーム全体の流れを分かりやすくしました  
- ログ表示を用いて、ハッキング風の演出を行いました 

【改善点・追加したい機能】
- スコアやランキング機能の実装  
- モバイル端末への対応
